# dayporesolver
Resuelve los daypo sin esfuerzo, primero debes usar https://github.com/lococoi/dayposcraper para obtener las preguntas y respuestas.
Luego debes abrir el py y seleccionar el archivo generado por el scraper, iniciar sesión (si querés guardar progreso, sino no) e ingresar al test y se resolverá solo.
Podes jugar con el tiempo para que no sea tan poco el tiempo que toma completarlo, en 0.01 en 124 preguntas tarda aprox 28 segundos.
